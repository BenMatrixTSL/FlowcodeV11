Flowcode11

Development files for Flowcode 11, components, device definitions (FCD) and code abstraction layer (CAL)

CAL folder contains C code specific to the various hardware platforms, common files include support for IO, ADC, DAC, I2C, SPI, UART, PWM, H-Bridge, I2S, Type definitions, Strings, Floats. etc.

Components folder contains a release directory which is the exported components currently in the public library system. A nonrelease folder contains private works in progress as well as internal exported components that are used in house but either niche or not polished enough to release to the public. Also in the components folder is a spreadsheet detailing the public components, the associated source project, as well as wiki page etc.

FCD folder contains all the various chip definitions as well as compiler make files (using windows batch scripts).
